Поскольку события "смерть пилота" как такового сейчас нет в функции OnGameEvent то настроить парсер надо так:
crashed:Если есть активный полёт то, пишем потерю ЛА и смерть, завершаем активный полёт игрока
	Если нет активного полёта то, пишем только потерю ЛА
ejected:Если есть активный полёт, то завершаем активный полёт и пишем катапультирование
	Если нет активного полёта, то просто пишем катапультирование
joined: Если есть активный полёт то, пишем смерть и потерю ЛА, завершам активный полет
	Если нет активного полёта, то ничего не пишем.

После появления события pilot_dead - перенастроим.
